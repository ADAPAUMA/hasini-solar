# ☀️ Hasini Solar India Pvt Ltd — Customer Management System

## ✅ MongoDB Atlas already connected!
The `.env` file has your cloud database pre-configured.
Data is stored online — works from anywhere!

---

## 🚀 Quick Start (2 Steps only!)

### Step 1 — Backend Start చేయండి

```
cd hasini-solar/backend
npm install
npm start
```
✅ You will see: MongoDB connected + Server running on port 5000

### Step 2 — Frontend Start చేయండి (New Terminal)

```
cd hasini-solar/frontend
npm install
npm start
```
✅ Browser లో http://localhost:3000 తెరుచుకుంటుంది!

## Features
- Register / Login (JWT secure)
- Add / Edit / Delete Customers
- 4-Stage Status Tracking
- Search by Name, Village, Bank, Phone
- Table View + Cards View
- Stats Dashboard
- MongoDB Atlas Cloud Storage
- Solar Panel Animated Background

## 4 Status Stages
1. బ్యాంక్ నుండి మొదటి పేమెంట్
2. సైట్ Installation Completed
3. బ్యాంక్ నుండి రెండోవ పేమెంట్
4. Finally కస్టమర్ సైట్ Completed
